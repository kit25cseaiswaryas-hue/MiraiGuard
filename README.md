# MiraiGuard

## AI-Powered Web Framework for Early Detection and Defense Against IoT Botnet Attacks

### Problem Statement
IoT devices are vulnerable to Mirai botnet attacks that can compromise devices and launch large-scale DDoS attacks.

### Solution
MiraiGuard uses Machine Learning to analyze network traffic and identify suspicious behavior at an early stage.

### Features
- AI-based attack detection
- Network traffic analysis
- Detects Mirai botnet activity
- Google Colab compatible
- Simple and user-friendly interface

### Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

### How to Run
1. Open `MiraiGuard.ipynb` in Google Colab.
2. Run all cells.
3. Enter network parameters.
4. View the prediction.
